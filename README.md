# dsc-finemap
DSC for comparing fine-mapping methods.

## Benchmarks

### SuSiE paper benchmark

Please use command 

```
dsc susie_paper.dsc -h
```

to view available runs, and 

```
dsc susie_paper.dsc
```

to run the benchmark.

### SuSiE z-score benchmark

Please use command 

```
dsc susie_z.dsc -h
```

to view available runs, and 

```
dsc susie_z.dsc
```

to run the benchmark.
